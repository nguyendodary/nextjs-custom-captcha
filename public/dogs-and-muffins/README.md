# CAPTCHA image dataset

Place the following image files in this folder:

**Dogs (10 files):**
- `dog1.png` … `dog10.png`

**Muffins (13 files):**
- `muffin1.png` … `muffin13.png`

Use PNG (or JPG) images. The CAPTCHA will randomly show 9 of these in a 3×3 grid; users must select all dog images to pass.

You can use the [Dogs vs Muffins](https://www.kaggle.com/datasets/iamsouravbanerjee/dogs-vs-muffins) or similar dataset and rename files accordingly.
